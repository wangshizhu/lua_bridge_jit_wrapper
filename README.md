### 概述
--------------------------------------------------

本源码使用Openresty维护的[Luajit2][Luajit2]，同时使用[LuaBridge][luabridge]，宿主语言使用C++

### 版本要求
--------------------------------------------------

Luajit2版本：v2.1-agentzh

LuaBridge版本：2.5

CMake版本：3.16+

GCC版本：8.x+
